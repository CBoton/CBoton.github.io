Dataset

For this assignment, our team decided to use the 2017 US Census Dataset located on Kaggle's website. We found that this dataset had a lot of possibilities and questions to be answered.

The data is available online at: https://www.kaggle.com/muonneutrino/us-census-demographic-data

Interactive Design Decisions

We first considered creating a map of California where the user could click on each county to view the data, but we decided this would not depict the data very accurately. Our team settled on using an interactive graph of the top ten populous counties in California. The user can select from 23 statistics to be compared by county. We went this route because it provides comparable data between the top ten counties in California, showing the differences and similarities of each county.

The interactive d3 starter code we used can be found at: https://vizhub.com/curran/98ba4daacc92442f8d9fd7d91bfd712a

2nd Interactive Design

As were are considering moving outside of California we thought it would be helpful if we could narrow down the search based on the highest income earning areas across the country. This visualization shows a few things to help narrow that search. First off we have a color coded map of the United States which shows the states that have a higher or lower average income for the entire state. The only problem with this is that there are only 9 states with an average income greater than California. To help expand the search we decided to display the top three average income counties per state which you can see by hovering over the red dots on the map. On mouseover the tooltip will display the county name, state, and average income for the county. The circles are also to scale, so the bigger the circle, the higher the average income.
